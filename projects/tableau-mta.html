Calculated Fields
7 custom fields built from raw data
SUMTotal Daily Riders
Aggregates across all 6 transit mode columns (Subway, Bus, LIRR, Metro-North, Access-A-Ride, Bridges & Tunnels) from the Daily Ridership source into a single daily figure.
LODvs Last Week %
Period-over-period delta comparing today’s total ridership against the same day 7 days prior. Drives the ▲▼ arrow indicators on BAN cards.
FIXEDvs Target %
Compares actual daily values against hardcoded targets: 4M total riders, 850K Bridges & Tunnels, 4K Staten Island Railway, 4K avg transfers per 6 hrs.
DIVOMNY Adoption Rate
OMNY ridership ÷ total ridership — feeds the bullet bar chart against the 50% adoption target reference line.
SUMKPI % of MTD Target
YTD Actual ÷ YTD Target, aggregated at domain level (Safety, Service Reliability, Infrastructure) for the KPI overview text table.
DIVTransfer Rate per Station
Transfers ÷ Ridership at station granularity. Surfaces in the map tooltip alongside raw ridership, transfers, and OMNY usage per station.
BOOL7-Day Filter Boolean
Flags records within the last 7 days to constrain the sparkline to the correct window while keeping the full dataset available for other views.
Chart Types & Design Decisions
6 chart types · each selected against a principle
🔟
BAN Cards with Delta Indicators
4 big number cards for Total Riders, B&T Riders, SIR Riders, and Avg Transfers. Each shows both % vs last week and % vs monthly target.
▲▼ arrows + green/orange color encoding — preattentive attributes from class content. Immediate attention to off-target metrics.
📈
Small Multiples — 5 Borough Panels
5 independent line charts (Bronx, Brooklyn, Manhattan, Queens, Staten Island), each with their own y-axis and a shared 120K/6-hour reference line.
Independent axes prevent scale distortion across boroughs — Tufte principle. Enables like-for-like comparison without flattening low-volume boroughs.
📈
Sparkline with Fixed Axis
7-day ridership trend line with a fixed y-axis range of 3M–5M and a 4M daily target reference line.
Fixed axis shows meaningful variation instead of zero-based flatness — Tufte. Zero-based would compress the signal into noise.
🌎
Density Heatmap (Mapbox)
Geographic heatmap over NYC showing station traffic concentration using Mapbox tile layer and lat/long from the 126K-row subway dataset.
Density map over point map — shows concentration patterns, not just locations. Reveals corridor clusters invisible in a dot scatter.
▮
Bullet Bar Chart — OMNY Adoption
Horizontal bullet bar showing OMNY adoption % against a 50% reference line target. Tooltip reveals fare class breakdown.
Bullet bar over gauge chart — more information-dense, avoids the wasted space of a semicircle. Few’s alternative to gauges.
📋
KPI Text Table with Tooltip Drill-Down
Domain-level KPI summary (Safety 49%, Service Reliability 101%/92%, Infrastructure 131%) with tooltip revealing sub-indicator breakdown on hover.
KPI domains sorted by failure — Safety first. Direct attention to worst performer per Big Book of Dashboards principle.
